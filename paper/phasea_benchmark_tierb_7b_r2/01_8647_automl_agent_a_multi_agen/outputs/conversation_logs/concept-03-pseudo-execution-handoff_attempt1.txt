<ctrl60># Agent-Driven Manim Code Generation

A Python framework designed to generate, debug, and execute **Manim animations** using **Large Language Models (LLMs)**. The system implements a robust code correction loop to ensure that generated Manim code is runnable and logically sound based on user-provided prompts.

## Features
- **Prompt-Based Animation Generation**: Converts high-level JSON or text descriptions into complex Manim scenes.
- **Automated Debugging Loop**: Automatically catches compilation/runtime errors and sends them back to the LLM for correction.
- **Pre-configured LLM Tutors**: Specialized system prompts designed to guide LLMs (like GPT-4, Claude, or local models) in writing valid Manim code.
- **Reference Pattern Support**: Includes context-aware examples for animations, graphs, and updaters to improve LLM accuracy.
- **JSON Metadata Integration**: Handles structured input including narration (voice), goals, and formulas.

## Architecture
1. **Instruction Set**: A set of critical rules (e.g., preserving original ideas, fixing specific errors).
2. **Reference Bank**: Standard Manim code snippets for common tasks (Transform, ValueTrackers, Plotting).
3. **Execution Engine**: Attempts to compile the Python script using `manim`.
4. **Error Handler**: Captures the traceback and formats it for the "Tutor" LLM.
5. **Correction Cycle**: The LLM outputs a corrected version until the scene renders successfully.

## Quick Start

### Prerequisites
- Python 3.8+
- [Manim](https://docs.manim.community/en/stable/installation.html) (Community Edition)
- FFmpeg
- An LLM API Key (OpenAI, Anthropic, etc.) or a local LLM setup.

### Setup
# Clone the repository
git clone https://github.com/yourusername/agent-manim-gen.git
cd agent-manim-gen

# Install dependencies
pip install -r requirements.txt

### Usage
Run the main generation script with your prompt:
python generate_animation.py --prompt "visualize binary search algorithm"

  "formula": "y = f(x)"
}

## How It Fixes Code
If a script fails with `NameError: name 'ValueTracker' is not defined`, the system prompt instructs the LLM:
1. "Output ONLY valid Python code."
2. "Fix the specific compilation/runtime errors reported in the log."
3. "Preserve the ORIGINAL VIDEO IDEA."

The LLM then returns the corrected imports or logic, which is re-tested by the engine.
